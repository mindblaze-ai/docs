# Access Tools in Swan Accounts

<Callout type="info">
Navigate to the Tools section in Swan Accounts to access account management utilities.
</Callout>

### Step 1 — Click the Desk link

Click the **Desk** link to navigate to the main dashboard. This is your starting point for accessing Swan's core features.

### Step 2 — Click the Accounts link

Click the **Accounts** link to enter the accounts section. This area contains all account-related functions and settings.

<CaptureImage src="./assets/access-tools-in-swan-accounts/step-02.jpg" alt="Step 2" render="eyJoaWdobGlnaHQiOnsieCI6MC4wMDcxNjIwNDExODE3MzY3OTUsInkiOjAuMjUzMDk0OTEwNTkxNDcxOCwid2lkdGgiOjAuMjE0ODYxMjM1NDUyMTAzODQsImhlaWdodCI6MC4wMzg1MTQ0NDI5MTYwOTM1NH19" />

### Step 3 — Click the Tools link

Click the **Tools** link to access the tools panel. This section provides utilities and features for managing your accounts.

## Summary

You have navigated to the Tools section within Swan Accounts and can now access the available account management tools.
