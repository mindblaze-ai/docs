# Navigate to Developer Setup Guide in LivingContext

<Callout type="info">
Navigate to the Developer Setup Guide section in LivingContext, the living documentation platform for Salesforce.
</Callout>

### Step 1 — Click the Business Process Overview button

Click the **02-Business-Process-Overview** button. This button is part of the navigation structure that leads toward the Developer Setup Guide section.

### Step 2 — Click the Developer Setup Guide button

Click the **03-Developer-Setup-Guide** button. This button opens the Developer Setup Guide section where you can access setup and configuration documentation.

### Step 3 — Focus the browser window

Click on the page to ensure the browser window is in focus. This prepares the interface for any subsequent interactions or content viewing.

<CaptureImage src="./assets/navigate-to-developer-setup-guide-in-livingcontext/step-03.jpg" alt="Step 3" render="eyJoaWdobGlnaHQiOnsieCI6MCwieSI6MCwid2lkdGgiOjEsImhlaWdodCI6MX19" />

### Step 4 — Focus the browser window again

Click on the page once more to maintain focus on the browser window. This ensures the page is ready for you to read or interact with the Developer Setup Guide content.

You have navigated to the Developer Setup Guide section in LivingContext and the browser window is now focused and ready for use.
