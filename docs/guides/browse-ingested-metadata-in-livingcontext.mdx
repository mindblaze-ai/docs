# Browse Ingested Metadata in LivingContext

<Callout type="info">
Access the ingested metadata browser in LivingContext to view Salesforce metadata that has been synced into your workspace.
</Callout>

### Step 1 — Open the workspace navigation

Click the **Show workspace nav** button to reveal the workspace rail. This expands the sidebar where you can access main navigation options.

### Step 2 — Navigate to Home

Click the **Home** link in the workspace rail. This takes you to the main workspace view where you can access other sections.

<CaptureImage src="./assets/browse-ingested-metadata-in-livingcontext/step-02.jpg" alt="Step 2" render="eyJoaWdobGlnaHQiOnsieCI6MC4wMTA4MTA4MTA4MTA4MTA4MTEsInkiOjAuMjAxNTI2NzI5MjA1MTgyNSwid2lkdGgiOjAuMjQ4NjQ4NjQ4NjQ4NjQ4NjYsImhlaWdodCI6MC4wNDgzNDYwNTU5Nzk2NDM3N319" t="4152" />

### Step 3 — Open Settings

Click the **Settings** link in the workspace rail. This opens the settings area where you can configure your workspace and account.

<CaptureImage src="./assets/browse-ingested-metadata-in-livingcontext/step-03.jpg" alt="Step 3" render="eyJoaWdobGlnaHQiOnsieCI6MC4wMTA4MTA4MTA4MTA4MTA4MTEsInkiOjAuNTA2ODcwMjIxMjQyMzQ2NSwid2lkdGgiOjAuMjIzNDIzNDIzNDIzNDIzNDIsImhlaWdodCI6MC4wNDgzNDYwNTU5Nzk2NDM3N319" t="11361" />

### Step 4 — Go to Connections

Click the **Connections** link in the section sidebar under Your account. This displays connection settings where you manage integrations and data sources.

<CaptureImage src="./assets/browse-ingested-metadata-in-livingcontext/step-04.jpg" alt="Step 4" render="eyJoaWdobGlnaHQiOnsieCI6MC4yNTI5NzI5ODk0Njg5NjExMywieSI6MC44MDkxNjAzMDUzNDM1MTE1LCJ3aWR0aCI6MC4yNTUxMzUxNDYxMzI0NjA2LCJoZWlnaHQiOjAuMDM1NjIzNDA5NjY5MjExMTk1fX0=" t="18284" />

### Step 5 — Browse the ingested metadata

Click the **Browse ingested metadata** link in the Salesforce section. This opens the metadata browser where you can view and explore all Salesforce metadata that has been ingested into your workspace.

<CaptureImage src="./assets/browse-ingested-metadata-in-livingcontext/step-05.jpg" alt="Step 5" render="eyJoaWdobGlnaHQiOnsieCI6MC45MjE1NDg0MzQ0MjA3NDg5LCJ5IjowLjU4ODU0OTYyNjA4NTg5NzgsIndpZHRoIjowLjE2MTA0MTY3MjE2NTMyOTM5LCJoZWlnaHQiOjAuMDMwNTM0MzUxMTQ1MDM4MTY3fX0=" t="28884" />

## Summary

You have navigated to the metadata browser in LivingContext and can now view the Salesforce metadata that has been synced to your workspace.
